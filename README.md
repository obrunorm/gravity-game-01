# 🎮 Unity 2D Combat Prototype

A small **2D gameplay prototype built with Unity** focused on implementing core mechanics such as player movement, gravity inversion, combat, and enemy interaction.

This project was created as a learning exercise to explore **Unity physics, the new Input System, and basic combat mechanics**.

---

# 🚀 Features

### Player Mechanics

* Horizontal movement using **Rigidbody2D**
* Gravity inversion jump mechanic
* Character **flip based on movement direction**
* Animation control using **Animator**

### Enemy System

* Basic enemy with **health system**
* Receives damage from player attacks
* Destroyed when health reaches zero

# 🎮 Gameplay Overview

The player can:

* Move horizontally
* Invert gravity to jump to the opposite surface
* Attack enemies within range
* Interact with basic enemy entities that take damage and get destroyed

The goal of this project was to experiment with **core gameplay programming patterns in Unity**.
